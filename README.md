# Flipbook SaaS Reader

A web-based PDF flipbook reader built with Next.js and PageFlip.

## Features

- PDF rendering using PDF.js
- Page flip animation
- Zoom in / zoom out
- Dark mode
- Fullscreen mode
- Thumbnail preview
- Page navigation

## Tech Stack

Next.js  
React  
Tailwind CSS  
PDF.js  
PageFlip

## Installation

Clone the repository

```
git clone https://github.com/kaviyarajasingh/flipbook-saas.git
```

Go to frontend

```
cd frontend
npm install
npm run dev
```

Open browser

```
http://localhost:3000
```
